Definitions

A static documentation source. Examples include: Publications, printed documents, static electronic documents, snapshop versions of databases.

Usage instances or treatment of a name of an organism within a reference.

The subset of taxon name usage instances that involve code-goverened actions on scientific names. Examples: Establishment of names for new taxa, subsequent new combinations (ICNafp), lectotypifications, neotypifications, first-reviser actions (ICZN).

A subset of nomenclatural acts that represent the establishment of a name for a new taxon, in compliance with the relevent code.

The first usage of a particular scientific name within a reference, regardless of whether or naot it is established in complicance with the relevent code.
